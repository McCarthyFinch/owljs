setClassVars

exports.setClassVars = function ()

Initializes o variable

for every class C in O, where C has a label L, set

o.L' = C

where L’ is a “safeified” version of L (ie valid js symbol), with spaces turned to underscores and invalid characters removed

Note that this is particularly useful in a REPL.  A user can type “o.epi<TAB>” and is offered options such as “epithlium” or “epiblast”
